
## Sales Analysis Project
## Project Overview
This project analyzes sales performance using Python, SQL, and Power BI to uncover actionable insights for business decision-making.
The workflow includes data cleaning, exploratory data analysis (EDA), SQL-based data merging, and interactive dashboard creation to monitor key business metrics.

## Tools & Technologies Used

Python ‚Üí Data cleaning, preprocessing, and exploratory data analysis

Libraries: Pandas, NumPy, Matplotlib, Seaborn

SQL ‚Üí Merging tables, filtering, and data retrieval within Python

Power BI ‚Üí Creating interactive dashboards for visualization & KPI tracking

## KPIs Used in the Dashboard

Total Revenue ‚Äì Measures overall sales performance.

Total Profit ‚Äì Tracks business profitability.

Profit Margin % ‚Äì Shows efficiency in converting sales to profit.

Total Orders ‚Äì Indicates sales volume.

Revenue per Order ‚Äì Highlights average customer spending.

Revenue by Region ‚Äì Shows geographic contribution.

Profit Margin by Region ‚Äì Compares profitability across regions.

Top/Bottom Products by Revenue ‚Äì Identifies best and worst performers.

Top/Bottom Products by Profit Margin ‚Äì Highlights most/least efficient products.

Revenue & Profit by Sales Channel ‚Äì Analyzes performance of Wholesale, Distributor, and Export channels.

Monthly Revenue & Profit Trends ‚Äì Tracks seasonality in business.

Order Value Distribution ‚Äì Segments customers by spending tier.

Unit Price vs. Profit Margin ‚Äì Identifies high-margin pricing bands.
